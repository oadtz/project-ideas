üö® CRITICAL FINDING: LINE cannot do cold outbound ‚Äî model pivot required

üö® Critical Discovery: LINE Cold Outbound is IMPOSSIBLE

  • LINE API requires opt-in: Push messages can ONLY be sent to users who have already added your LINE Official Account as a friend. You CANNOT cold-message strangers.
  • No user discovery: There is NO way to find a LINE user by phone number or email and message them. The user must initiate the relationship.
  • Anti-spam enforcement: LINE explicitly prohibits mass messaging to same user, mass requests, and attempting to identify user attributes. Violations = account suspension.
  • Impact on thesis: The original positioning ("AI cold outbound via LINE where Thai B2B happens") is fundamentally broken. Must pivot to email-first cold outreach with LINE as warm engagement channel.
  • Score impact: Adjusted from 35/50 down to 33/50. Competition Gap reduced because email-first model is closer to what global AI SDRs already do.

üáπüá≠ Thai B2B Sales Culture Deep Dive

  • Relationship-first: 60-70% of Thai B2B comes from referrals. "‡πÄ‡∏õ‡πá‡∏ô‡∏Ñ‡∏ô‡∏£‡∏π‡πâ‡∏à‡∏±‡∏Å" (being someone who knows someone) is critical. Trust before transaction.
  • Cold outreach is culturally weak: Pure American-style cold email blasting will likely fail. Thai business culture values warm introductions over cold pitches. Expected response rate: 3-5% (vs 10-15% globally).
  • LINE usage: 53M Thai LINE users. LINE is used for B2B communication AFTER relationship established, not for cold outreach. LINE OA = customer engagement, not prospecting.
  • LinkedIn: ~5-6M Thai accounts but very low daily usage. Meaningful only for multinational/tech companies. Nearly irrelevant for Thai SME sales.
  • CRM adoption: Very low. Most Thai SMEs use LINE chats + Excel as "CRM." Salesforce = large enterprise only. HubSpot growing in mid-market. Local alternatives: R-CRM (Readyplanet), Wisible.
  • Price sensitivity: ‡∏ø9,900/mo is significant for most Thai SMEs. ‡∏ø15K/mo = mid-market+ only. Need ‡∏ø4,900/mo starter tier.

üì± LINE API Technical Analysis

  • Push messages: Can send to friends at any time. Counts against monthly quota. User ID required (obtained when user adds OA as friend).
  • Reply messages: Unlimited and FREE. Only triggered by user's incoming message.
  • Narrowcast: Segment friends by age, gender, OS, region. Useful for targeting warm leads WITHIN friend list.
  • Thailand pricing (estimated): Free: ~200-500 msgs/mo. Light: ~‡∏ø1,500-2,000/mo for 5,000 msgs. Standard: ~‡∏ø5,000-8,000/mo for 30,000+ msgs.
  • Rate limits: 100,000 requests/minute (generous). Up to 5 message objects per request.
  • Prohibited: Mass messages to same user, mass requests, identifying user attributes. Violations = suspension.
  • Verdict: LINE is excellent for warm engagement/nurturing. IMPOSSIBLE for cold outbound. Must pivot model.
